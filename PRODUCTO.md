# Costa Blanca Leads — Producto y Modelo de Negocio

## ¿Qué vendemos exactamente?

Vendemos **páginas web con SEO local** para inmobiliarias de **Calpe, Altea, Benidorm, Moraira, Jávea y toda la Costa Blanca**.

No es una web genérica. Es una **máquina de captar clientes** optimizada para que la inmobiliaria aparezca la primera en Google cuando alguien busca "comprar piso en Calpe", "inmobiliaria Altea", "chalet en venta Benidorm", etc.

## ¿Qué incluye cada plan?

### Setup único — 790€ (lanzamiento) / 1.290€ (habitual)
Incluye: diseño web, dominio, configuración SEO, Google Maps, WhatsApp integrado

### Básico (79€/mes)
- Landing page profesional con la marca de la inmobiliaria
- Catálogo de propiedades — cada propiedad tiene su propia página
- SEO local básico
- Formulario de contacto + WhatsApp
- Diseño responsive

### Profesional (149€/mes) — Más popular
- Todo lo del Básico
- Nuevas páginas SEO mensuales (una página por propiedad o zona)
- SEO local avanzado + Google Maps
- Blog SEO integrado
- Optimización de velocidad
- Informes mensuales

### Premium (249€/mes)
- Todo lo del Profesional
- Contenido SEO mensual continuo
- Gestión de Google Business Profile
- Soporte prioritario 24/7

## ¿Cómo se entrega?

### Para el cliente
1. El cliente nos da: nombre, logo, fotos de propiedades, descripciones
2. Nosotros creamos la web con su marca (48h máximo)
3. El cliente tiene su web online con su propio dominio
4. Cada vez que tiene una propiedad nueva, nos la envía y la subimos

### Para ti (cómo lo automatizas)
- Tienes una plantilla base (la demo en `/demo`) que personalizas por cliente
- Cambias: logo, colores, nombre, propiedades, textos
- El dominio lo compras desde Namecheap/Dondominio (~10€/año) y lo apuntas a Vercel
- El SEO se configura una vez y funciona solo

## ¿Por qué le interesa a una inmobiliaria?

1. **Mercado competitivo** — Hay 30+ inmobiliarias solo en Calpe. Google es el campo de batalla.
2. **Google es el canal #1** — El 90% de compradores empieza por Google. Sin web, no existen.
3. **Su web actual es mala** — Webs lentas, sin SEO, sin catálogo. Fácil de superar.
4. **Precio ridículo para ellos** — Una comisión media es 5.000-15.000€. 149€/mes es nada.
5. **Sin permanencia** — Riesgo cero para ellos.
